My Links - React

What is it

Its a site were you can add web links to a list. You can create, edit and delete links.


Why was it built

To practice using React with a Node.js backend api and saving the data into mongodb. Also to use React Router to have multiple pages in React.

Github

Technologies Used

Node.js

Used to build an api that the react application can intract with to get and post data.

Mongodb

Used to store the links created

React

Used to build the frontend of the application
