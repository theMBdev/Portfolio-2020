Note Timer

What is it

Its a note taking application with time taking functionality. The idea was to track how much time you spend on different things. The note timeline is ordered by each day and split by each different notebook.


Why was it built

Built for personal use and to see how database relationships work in mongodb. Also for user interface design practice.
