Map Markers

What is it

A web application that lets users view the location of landmarks on a world map. The users browse the map and can click on a marker to see photos of what's there. The user can submit new locations and photos to the map after signing up. Each submission has to be verified by the site admin before it is added.


Why was it built

Built to practice with node js. Having built a blog in node js before this I wanted to build something new and a little more complex. I had worked with maps before so decided to create a project that would combine maps with node. Last time I used used google maps so this time I used another provider (leaflet/mapbox).
Using cloudinary i also got experience of uploading images to a site and storing the url to access the correct images in the dataabse. As this site has a sign up and an admin section i also worked with user authentication and roles.
